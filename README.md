# 🛒 ShopMate

An E-Commerce Web Application developed using Spring Boot, JSP, Hibernate, JPA, MySQL, HTML, CSS and Bootstrap.

## 🚀 Features

- User Registration & Login
- Product Catalog
- Add to Cart
- Remove from Cart
- Customer Profile
- Shopping Cart
- Bill Generation
- Order Confirmation
- Responsive UI

## 🛠️ Technologies Used

- Java 21
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- JSP
- JSTL
- MySQL
- HTML5
- CSS3
- Maven

## 📂 Project Structure

```
src
 ├── controller
 ├── model
 ├── repo
 ├── service
 ├── templates
 ├── static
 └── application.properties
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/shilimkar-ajinkya7/ShopMate.git
```

2. Open in STS/Eclipse

3. Configure MySQL database

4. Update application.properties

5. Run ShopmateApplication.java
